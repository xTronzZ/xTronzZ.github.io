GitHub Access

Publishing requires a GitHub Personal Access Token with contents:write on this repository. The token is stored in this browser only (localStorage) and never sent anywhere except api.github.com.

Create a fine-grained token at github.com/settings/personal-access-tokens: access xTronzZ.github.io, repository permission Contents: Read & write.
